Conectando ao GitHub

-Primeiramente, é necessário criar um novo repositório no GitHub.

-Para conectar o projeto da máquina com a núvem do github é necessário entrar no diretório do projeto pelo cmd e digitar o seguinte comando:

  • git remote add origin https://github.com/tierres/aprendizado-git-github.git

-O link é disponibilizado na página do novo repositório no GitHub

-Assim o projeto da máquina estará conectado com o projeto na nuvem.
