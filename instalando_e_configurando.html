Instalando e Configurando o Git

  1. Primeiramente baixe o git em git-scm.com/downloads e instale-o
  2. No caso de usuários de windows, abra o aplicativo "Git Bash" para poder utilizar comandos de linux, visto que a maioria dos sevidores rodam em linux.
  3. Ao abrir o Git Bash configure seus dados utilizando os seguintes comandos:
  4. git config --global user.name "Tierres Griep"
  5. git config --global user.email "tierresgriep@gmail.com"
  6. Após configurar o git é só navegar pelas pastas do sistema até chegar no projeto onde desejas rodar o git. Utilize git init para iniciar um repositório git.
